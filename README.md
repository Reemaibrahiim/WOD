# 🎁 WOD
## Course Project: IT320 Practical Software Engineering 
It is an interactive social gifting platform designed to transform the gift-giving experience into an organized, collaborative, and stress-free process. The platform enables users to create wishlists, share gift ideas, coordinate group contributions, reserve gifts to prevent duplication, and manage important occasions through an integrated calendar

Unlike traditional wishlist or delivery applications that focus only on purchasing, WOD emphasizes social connection and organization—turning gift-giving into a thoughtful and collaborative experience.

## 🛠️ Technologies Used
Frontend: HTML5, CSS3, JavaScript (Progressive Web App)

Backend: PHP

Database: MySQL

Version Control: Git & GitHub

Project Management: Jira (Agile/Scrum)

## 🚀 Features
User registration, login, and logout with secure authentication

Create, edit, and manage personal wishlists with visibility controls

Browse and interact with posts on the Explore feed

Reserve gifts from friends' wishlists to prevent duplication

Group gifting with contribution tracking and transparent coordination

Search for friends and view their profiles and wishlists

Integrated calendar for adding and tracking important occasions

Like and save posts to wishlist directly from the feed

Store accounts with external shop links

## 💡 Key Features:
Users register and log in securely to access their personalized dashboard

Create posts with images and descriptions to share gift ideas

Add posts directly to personal wishlists from the Explore feed

Reserve available items on friends' wishlists to prevent duplicate gifts

Start or contribute to group gifts with clear payment tracking

Control wishlist visibility (public, friends-only, or hidden)

Add occasions to calendar to never forget important events

Search for friends and view their public profiles and wishlists

Like posts (anonymously) to show appreciation
